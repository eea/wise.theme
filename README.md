wise.theme
============

Theme package for WISE


Installation
------------

* Install ``wise.theme`` by adding it to your buildout:

    ```
    [buildout]
    ...
    eggs =
        ...
        wise.theme
    ```

* And then running:

    ``bin/buildout``
